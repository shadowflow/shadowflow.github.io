文本处理命令

2016.09.23 BluekyZhao

一、排序：sort

-d            按照字典顺序（只按照字母数字空格排序）
-f            忽略大小写
-u            删除重复行（和uniq不同，它按照要比较的字段，删除整行）
-t: -k3     以：为分隔符，按照第三个字段排序

二、去重：uniq

-c          在每行前面加上该行出现的次数
-d             仅显示重复行
-u             仅显示不重复的行

三、统计：wc

-l            显示文件的行数
-w             字数
-c          字符数

四、取出：cut

-d            指定分隔符
-f             取出第几列
example:
cut -d ':' -f 1,6,7 /etc/passwd

五、连接：join

根据key合并文件
join 1.txt 2.txt

六、替换：tr

tr 'a-z' 'A-Z' <1.txt >2.txt    #把1.txt中的小写替换成大写

七、比较：vimdiff

vimdiff 1.txt 2.txt

这是基本的文本处理，还有更厉害的awk,但是他几乎成了一门单独的语言。请单独学习
