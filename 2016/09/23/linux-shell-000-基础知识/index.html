    1:变量

    杂项注意

    1. source file #不创建新的进程，通常用来使新修改的脚本立即生效
    2. “” #弱引用
    3. ‘’ #强引用
    4. 函数里面局部变量加local，否则就是全局变量

      环境变量

    5. export PATH=./:$PATH (直到本shell消亡）
    6. env -i ./:$PATH hehe.sh (本条命令临时作用)

    2: 参数

    符号 含义
    * 用一个单字符串显示所有向脚本传递的参数
    @ 用多个词显示所有向脚本传递的参数
    # 显示参数的数量
    $ 当前进程的ID号
    ! 后台运行的最后一个进程的ID号
    ? 最后命令的退出状态，只有0表示正确

    3: 特殊文件

    ###/dev/null ###
    黑洞，进去的全没了，能干让终端闭嘴之类的事

    ###/dev/zero ###
    额，创建指定大小，并且初始化为空的文件，极少用

    /dev/tty

    进到这里的东西全都被重定位到显示器上
